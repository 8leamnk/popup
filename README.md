# 팝업 훈련

## STEP1

1. 사용자가 홈에 진입한다. 이 때 팝업이 뜨지 않는다.
2. 다른 페이지로 이동한다.
3. 다시 홈으로 돌아온다. 이 때 팝업이 뜬다.

## STEP2

1. 사용자가 홈에 진입한다. 이 때 팝업이 뜨지 않는다.
2. 다른 페이지로 이동한다.
3. 다시 홈으로 돌아온다. 이 때 1번 팝업이 뜬다.
4. 다른 페이지로 이동한다.
5. 다시 홈으로 돌아온다. 이 때 2번 팝업이 뜬다.
6. 다른 페이지로 이동한다.
7. 다시 홈으로 돌아온다. 이 때 3번 팝업이 뜬다.
8. 10번 팝업이 뜰 때까지 반복 된다.
